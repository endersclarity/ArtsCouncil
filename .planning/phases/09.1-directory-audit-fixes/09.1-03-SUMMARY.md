- B2: All 5 bare `getElementById` calls in bindings.js wrapped in null guards (`if (el)` before `.addEventListener()`) — eliminates TypeError crash on pages where those elements don't exist
- S5: All ES6 syntax removed from bindings.js — zero `const`, `let`, or `=>` occurrences; 100% ES5 var/function syntax
- S1: Full venue/layer data chain complete:
  - `events-view.js` `getEventsCardsHTML`: passes `event.venue_name || event.location || ''` and `event.category || ''` as 4th/5th args to `renderEventBookmarkButton`
  - `events-view.js` `getEventsRowsHTML`: same — both call sites now emit `data-event-venue` and `data-event-layer` HTML attributes on rendered bookmark buttons
