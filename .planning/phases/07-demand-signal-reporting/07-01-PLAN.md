In `index-maplibre-analytics.js`:

1. Add `_getSessionHash()` helper INSIDE the IIFE, before the `track()` function:
```javascript
function _getSessionHash() {
  var key = 'gvnc_session_hash';
  var existing = sessionStorage.getItem(key);
  if (existing) return existing;
  var hash = Date.now().toString(36) + Math.random().toString(36).slice(2, 8);
  sessionStorage.setItem(key, hash);
  return hash;
}
```

2. Modify the `track()` function: after the dedup check passes and before `window.umami.track()`, inject session_hash into the payload:
```javascript
var payload = Object.assign({ session_hash: _getSessionHash() }, data || {});
window.umami.track(eventName, payload);
```
Replace the existing `window.umami.track(eventName, data || {});` line with the two lines above.
