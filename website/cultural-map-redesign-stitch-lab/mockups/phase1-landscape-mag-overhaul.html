Direction

This pass drops Pinterest motifs and uses the travel-magazine structure as the primary layout model: cover -> contents -> spread modules -> index.

Reusable Modules

Each module can map to existing app pieces: category tabs, event rails, map, editorial cards, and directory list without changing data contracts.

Why This Works

The value is hierarchy and pacing, not mountain/lake imagery. Photography can be swapped while the editorial skeleton remains intact.
