## Recommended Use in Website Mockups
- Use 4-6 of these as rotating feature modules.
- Keep image crops bold; avoid tiny thumbnails.
- Pair each with one short line of narrative copy and one CTA.
- Maintain one shared visual system (type, spacing, accents) so modules feel unified.
