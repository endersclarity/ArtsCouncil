Phase 1: Design & Visual Refresh

Before / After Comparison Mockup

What Changes
Background: Warm parchment (#F4EFE6) → Clean near-white (#FAFAF8)
Text: Slate navy (#4A6B7C) → True dark (#1a1612) for high contrast
Shadows: Dual-direction neumorphic embossing → Flat editorial drop shadows
Cards: Shared token system (radius, shadow, border, hover) across all card types
Map border: 4px heavy neumorphic frame → 1px subtle or borderless
Copy: "Cultural Asset Map" → "Explore Nevada County" / visitor-facing language
Category colors: Muted heritage tones → Brighter, more saturated accents

Design Token Changes
Token Before After Why
--neuro-bg / --cream #F4EFE6 → #FAFAF8 Clean white like MUSE pages
--ink #4A6B7C → #1a1612 True dark for 16:1 contrast
--gold #D4AF37 → #c8943e Slightly warmer, less brassy
--shadow-raised 6px 6px 12px (dual-direction) → 0 2px 8px (flat drop) Editorial, not embossed
--cat-landmarks #A0522D → #c0392b Brighter, more saturated
--cat-arts #5C7A5C → #27ae60 Brighter, more vibrant
--cat-cultural #4A6B7C → #2980b9 Stronger blue, not washed out
--cat-galleries #7A6B7D → #8e44ad Rich purple, more distinct
What Stays the Same
Typography: Playfair Display / DM Sans / JetBrains Mono (already good)
Layout structure: Hero tabs, map, editorial cards, explore list, colophon (all preserved)
Tabbed discovery panel: Same interaction model, cleaner visual treatment
Watercolor icons: Kept at 32px in category pills (reduced prominence elsewhere)
All id/aria-*/data-* attributes: Untouched (deep links and accessibility preserved)
CSS class names: Unchanged (36+ JS modules depend on them)
