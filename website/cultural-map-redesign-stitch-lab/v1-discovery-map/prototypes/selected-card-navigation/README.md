# Selected Place Navigation Prototype

Throwaway UI route for testing how a V1 Discovery Map selection from the Directory Browser should reveal a Unified Place Card while preserving browse context.

Variants:

- `?variant=replace`: Replace With Back Trail
- `?variant=split`: Split Stack
- `?variant=drawer`: Card Drawer With Context Rail
- `?variant=tabs`: Breadcrumb Tabs

This route uses static mock data only. It is not wired into production data or source records.
